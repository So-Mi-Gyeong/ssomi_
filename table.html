TABLE

테이블이란 여러 종류의 데이터를 보기 좋게 정리하여 보여주는 표를 의미

  • caption - caption, 테이블이나 사진, 삽화 등에 붙는 설명을 정의할 때 사용.
    table 요소는 단 하나의 caption 요소만을 명시할 수 있으며, caption 요소는 언제나 table 요소 바로 다음에 위치해야 한다.
    caption 요소의 콘텐츠는 테이블 바로 위쪽에 가운데 정렬되어 표시되지만,
    css의 text-align이나 caption-side 속성을 사용하여 캡션의 위치나 정렬 방법 등을 변경할 수 있다.
    html5 에서는 caption태그의 align 속성을 지원하지 않는다.
  • scope - th태그의 scope 속성은 해당 헤더 셀이 관련되는 셀의 종류를 명시한다.
    일반 웹 브라우저에서는 th 요소에 scope 속성을 명시해도 시각적 효과가 나타나지 않지만, 스크린 리더와 같은 장치에서는 유용하게 사용된다.
  • colgroup - 해당 셀이 열의 그룹을 위한 헤더 셀임을 명시한다.
  • colspan - colspan 속성을 사용하면 테이블의 열(column)을 합칠 수 있다.
  • col - 해당 셀이 열(column)을 위한 헤더 셀임을 명시한다.
  • rowgroup - 해당 셀이 행의 그룹을 위한 헤더 셀임을 명시한다.
  • rowspan - rowspan 속성을 사용하면 테이블의 행(row)을 합칠 수 있다.
  • row - 해당 셀이 행(row)을 위한 헤더 셀임을 명시한다.
  • id / headers - 해당 데이터 셀과 연관된 헤더 셀(header cell)들의 id 속성값을 공백으로 구분한 리스트
  • tr - 열을 구분
  • th - 열의 제목을 나타내며, 모든 내용은 자동으로 굵은 글씨에 가운데 정렬이 된다.
  • td - 열을 각각의 셀(cell)로 나누어 준다.

table에서 border 속성값을 따로 명시하지 않으면, 해당 테이블은 언제나 빈 테두리를 가지게 된다.
테이블의 테두리(border)가 두 줄씩 나타나는 이유는 table태그와 th태그, td태그가 모두 자신만의 테두리를 가지고 있기 때문이다.
두 줄로 표현되는 테두리를 한 줄로 설정하려면 border-collapse 속성을 사용해야 한다.
border-collapse 속성값을 collapse로 설정하면, 테이블의 테두리를 한 줄로 표현할 수 있다.
ex - table, th, td {border: 1px solid #000; border-collapse: collapse;}
